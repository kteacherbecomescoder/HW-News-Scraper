# Mongo News Scraper

Mongo News Scaper is web application that scrapes news from New York Times web site. 


## See it live here:   https://hw-news-scraper.herokuapp.com/


### Pre-requisites

`Install Node.js`

### Technologies used:
* node.js 
* Express.js 
* Bootstrap
* mongoose
* request
* cheerio
* handlebars
